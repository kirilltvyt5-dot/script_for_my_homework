Шаги создания скрипта:  
Выбор редактора. Можно использовать любой текстовый редактор, например Nano, Vim, VS Code.  
Создание файла. В терминале можно создать файл с помощью команды   
  
touch  
  
, указав имя файла с расширением   
  
.sh  
  
.  
Написание кода. В первую строку файла обязательно добавляется shebang —  указание системы, какой интерпретатор использовать для выполнения скрипта. Для Bash это выглядит так:   
  
#!/bin/bash  
  
. После этого можно писать команды, которые обычно вводите в консоли  
Сохранение и выход из редактора. Например, в Nano для сохранения нужно нажать   
  
Ctrl + O  
  
, затем   
  
Enter  
  
, а для выхода —   
  
Ctrl + X
