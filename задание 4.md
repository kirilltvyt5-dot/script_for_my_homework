Инструкция по отправке задания:  
Инициализируйте локальный репозиторий командой   
  
git init  
  
.  
Добавьте файл задания с помощью   
  
git add задание.md  
  
.  
Сохраните изменения с комментарием   
  
git commit -m "Задание выполнено"  
  
.  
Отправьте изменения в удалённый репозиторий   
  
git push origin main  
  
.  
Важно: убедитесь, что у вас есть права на редактирование репозитория, и вы используете корректное соединение (HTTPS или SSH), если это требуется.
